autojump:瞬移神器 j +关键词

安装教程：

  • sudo apt-get install autojump
  • vim .zshrc 加上[[ -s ~/.autojump/etc/profile.d/autojump.sh ]] && . ~/.autojump/etc/profile.d/autojump.sh
  • centos需要额外安装autojump-zsh
  • 编辑~/.zshrc：添加plugins=(git autojump)
  • 在使用j xxxx前,要先cd xxx/xxxx一次
  • 查看记录/配置文件: j --stat

  • source ~/.zshrc重新编译zsh
